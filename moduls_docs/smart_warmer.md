# 🔥 Умный прогрев (Smart Warmer)

Этот модуль предназначен для «оживления» сессий Telegram и повышения уровня доверия (trust) со стороны антифрод-систем мессенджера. Имитация реального поведения пользователя снижает риск блокировки при последующих рассылках или инвайтинге.

### 📝 Основные функции
1.  **Вступление в чаты**: Бот переходит по предоставленным ссылкам и присоединяется к группам. Порядок вступления рандомизирован.
2.  **Эмуляция чтения**: Модуль открывает последние 5–10 сообщений в каждом чате и делает паузы между ними (1–3 секунды), имитируя прокрутку ленты реальным человеком.
3.  **Взаимодействие с контентом**: С шансом 20% бот выбирает случайный пост из последних 20 сообщений чата и пересылает его в свои «Избранные сообщения» (Saved Messages).
4.  **Умные задержки**: После обработки каждого чата модуль делает паузу, длительность которой выбирается случайно в заданном пользователем диапазоне.

### ⚙️ Параметры модуля
- **Ссылки на чаты**: Список публичных или приватных ссылок на группы (по одной в строке). Рекомендуется использовать «белые» чаты без спама.
- **Мин. задержка**: Минимальное время ожидания между действиями (в секундах). Рекомендуется от 30 секунд для новых аккаунтов.
- **Макс. задержка**: Максимальное время ожидания. Рекомендуется до 120–300 секунд.

### 💡 Советы по использованию
- Не добавляйте слишком много чатов (более 5–10) за один проход для одного аккаунта.
- Используйте разнообразные чаты: новостные каналы, группы по интересам, тематические сообщества.
- Рекомендуется запускать прогрев раз в 2–3 дня для поддержания активности сессии.
