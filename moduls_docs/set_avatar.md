# 🖼️ Установить аватарку (Set Avatar)

Этот модуль предназначен для быстрой установки или смены фотографии профиля для выбранного аккаунта. Использование этого модуля экономит время при оформлении новых сессий.

### 📝 Основные функции
1.  **Смена фото профиля**: Модуль загружает выбранное изображение из вашего компьютера и устанавливает его на аватарку Telegram.
2.  **Обновление в UI**: Фотография профиля в главном окне ShadowGram обновится автоматически сразу после завершения работы плагина.
3.  **Безопасная инициализация**: Модуль выполняет дополнительную процедуру «прогрева» сессии (`client.start()`), чтобы избежать ошибок при смене личных данных.

### ⚙️ Параметры модуля
- **Выбрать изображение**: Путь к файлу на вашем диске. Поддерживаются форматы JPG, PNG, JPEG. Рекомендуется использовать квадратные изображения размером 512x512 и более.

### 💡 Важные особенности
- В этом модуле действует ограничение: **можно выбрать только один аккаунт за раз**. Это сделано для того, чтобы случайно не установить одну и ту же фотографию на все ваши аккаунты одновременно.
- При клике на другой аккаунт в списке выбора, галочка с предыдущего снимется автоматически.

### 🛡️ Советы
- Используйте разнообразные фотографии для разных аккаунтов, чтобы избежать подозрений со стороны антифрод-систем Telegram.
- Рекомендуется устанавливать аватарку не сразу после входа в аккаунт, а через 15–30 минут «отлежки».
