# 🛡️ Privacy Guard (Защита приватности)

Этот модуль предназначен для быстрой настройки параметров безопасности и конфиденциальности новых аккаунтов. Позволяет одним кликом скрыть информацию о пользователе и закрыть доступ к аккаунту для посторонних.

### 📝 Что изменяет этот модуль
1.  **Номер телефона (Phone Number)**: Устанавливает видимость номера телефона — **«Никто» (Nobody)**.
2.  **Поиск по номеру (By Phone Number)**: Ограничивает поиск аккаунта по номеру телефона только **«Мои контакты» (My Contacts)**.
3.  **Приглашения в группы (Group Invites)**: Позволяет добавлять вас в чаты только **«Моим контактам» (My Contacts)**.
4.  **Время захода (Last Seen & Online)**: Скрывает статус вашего последнего входа для всех — **«Никто» (Nobody)**.
5.  **Звонки (Voice Calls)**: Разрешает звонить на этот аккаунт только **«Моим контактам» (My Contacts)**.

### ⚙️ Особенности работы
- Модуль работает через низкоуровневые вызовы Telegram API (`Invoke`), что гарантирует корректное применение настроек в официальном приложении.
- После успешного выполнения настроек в логах появится подтверждение «Все настройки приватности успешно применены!».

### 💡 Советы по использованию
- Запускайте этот модуль сразу после регистрации нового аккаунта или добавления его в систему.
- Это эффективная защита от «спама» в личные сообщения и нежелательных приглашений в группы.
- Настройки приватности не влияют на работу других модулей автоматизации ShadowGram.
