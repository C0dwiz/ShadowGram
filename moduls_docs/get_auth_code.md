# 🔑 Получить код (Get Auth Code)

Специализированный модуль для получения 5-значного кода подтверждения от официального аккаунта Telegram (ID: 777000). Позволяет использовать текущую сессию для входа в этот же аккаунт на других устройствах (например, в мобильное приложение или Telegram Desktop).

### 📝 Основные функции
1.  **Поиск в истории**: Сначала модуль просматривает последние 3 сообщения от Telegram (777000). Если код уже был прислан недавно, он будет мгновенно извлечен.
2.  **Ожидание в реальном времени**: Если кода нет в истории, модуль переходит в режим ожидания (по умолчанию 60 секунд). Он постоянно опрашивает входящие сообщения от Telegram.
3.  **Извлечение регулярным выражением**: Бот автоматически находит 5 цифр кода внутри текста сообщения и выводит их крупным шрифтом в консоль.

### ⚙️ Режим работы
- **Время ожидания**: 60 секунд. Если за это время код не придет, модуль завершит работу с предупреждением.
- **Безопасность**: Модуль только читает входящие сообщения и ничего не отправляет.

### 💡 Как использовать
1.  Запустите вход в аккаунт на новом устройстве (введите номер телефона).
2.  Дождитесь, когда Telegram предложит «введите код, отправленный в приложение на другом устройстве».
3.  Запустите этот модуль в ShadowGram.
4.  Скопируйте полученный код из консоли логов и введите его на новом устройстве.

### 🛡️ Предостережение
- Не запрашивайте коды слишком часто, это может привести к временному ограничению на получение SMS или сервисных кодов со стороны Telegram.
