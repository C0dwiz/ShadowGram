## 🛠 Новые возможности системы

### ⚡ Параллельное выполнение и Ротации
Некоторые модули (например, «Просмотры») поддерживают **параллельный запуск**. Это означает:
- Все выбранные аккаунты работают **одновременно**.
- **Staggered Start**: Система автоматически добавляет случайную задержку (от 1 до 120 минут) перед стартом каждого аккаунта, чтобы избежать подозрений со стороны Telegram.
- **Бесконечный цикл (Ротации)**: После завершения задачи аккаунт уходит в «сон» на 3–6 часов, после чего автоматически начинает новый цикл работы.

### 🚀 Окно активных задач
Для мониторинга параллельных процессов используется отдельное окно:
- Каждая запущенная пачка задач открывается в **новой вкладке**.
- **🛑 Кнопка остановки**: Вы можете принудительно завершить работу любого модуля в любой момент. Прокси-туннели будут закрыты корректно.

### 🖱 Удобство навигации
- **Горизонтальный скролл**: Зажмите клавишу **Shift** и крутите колесико мыши для быстрой прокрутки списков и логов вправо или влево.

## 📝 Разработка своего плагина

Если вы хотите создать свой модуль, ознакомьтесь с базовым классом `src/core/base_module.py`. 
- Установите `ALLOW_PARALLEL = True`, чтобы включить режим одновременной работы аккаунтов и ротации.
- Установите `SINGLE_ACCOUNT = True`, если модуль требует работы только с одной сессией за раз.
