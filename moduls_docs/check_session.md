# 🔍 Проверка сессии (Check Session)

Основной диагностический модуль системы ShadowGram. Предназначен для быстрой проверки работоспособности сессий без необходимости заходить в них вручную через клиент.

### 📝 Основные функции
1.  **Детектирование банов**: Модуль мгновенно распознает заблокированные аккаунты (User Deactivated) и вылетает с ошибкой «Аккаунт в БАНЕ».
2.  **Проверка авторизации**: Проверяет, валидна ли сессия (AuthKeyUnregistered / Unauthorized). Если сессия «вылетела», вы получите соответствующее уведомление.
3.  **Получение данных профиля**: Бот запрашивает информацию о текущем пользователе (Username, ID) и отображает её в логах.
4.  **Загрузка аватара**: Если у аккаунта есть фотография профиля, модуль скачивает её в папку профиля (`avatar.jpg`), и она тут же появляется в круглом поле главного списка аккаунтов.

### ⚙️ Особенности работы
- Модуль использует прокси-туннель `Gost`, если прокси указан в настройках аккаунта.
- Безопасная проверка: не требует ввода номера телефона или подтверждающего кода.

### 💡 Рекомендации
- Запускайте этот модуль сразу после добавления новой сессии в папку профиля.
- Рекомендуется проверять аккаунты перед запуском «Умного прогрева» или других активных плагинов.
- Если аккаунт помечен как «АКТИВЕН», вы увидите его Telegram ID и Username.
